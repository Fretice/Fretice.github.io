1. 安装软件

在Ubuntu中安装软件的几种方式:

  • 通过自带的软件中心安装软件
  • 通过 apt install 直接安装
  • 通过 dpkg -i .deb 安装 deb文件

2. 卸载软件

在Ubuntu中卸载软件的几种方式:

  • 通过自带的软件中心卸载
  • 通过 apt remove 卸载
    1. 直接卸载同时保留应用配置: apt remove
    2. 卸载并清空应用配置: apt –purge remove
